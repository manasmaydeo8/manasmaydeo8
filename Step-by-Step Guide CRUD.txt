Step-by-Step Guide CRUD
CRUD Program in NetBeans (Java + Database)
1️⃣ Install Required Software

Before starting the practical, make sure these are installed.

Apache NetBeans IDE

JDK (Java Development Kit)

MySQL Server

XAMPP or MySQL Workbench (optional for database management)

2️⃣ Create Database in MySQL

Open phpMyAdmin / MySQL Workbench

Run this SQL:

CREATE DATABASE studentdb;

USE studentdb;

CREATE TABLE student(
id INT PRIMARY KEY AUTO_INCREMENT,
name VARCHAR(50),
email VARCHAR(50),
course VARCHAR(50)
);

This table will store student records.

5️⃣ Create Database Connection Class

Create new class:

DBConnection.java

Code:

import java.sql.Connection;
import java.sql.DriverManager;

public class DBConnection {

    public static Connection getConnection(){

        Connection con = null;

        try{

            Class.forName("com.mysql.cj.jdbc.Driver");

            con = DriverManager.getConnection(
                    "jdbc:mysql://localhost:3306/studentdb",
                    "root",
                    ""
            );

        }catch(Exception e){
            System.out.println(e);
        }

        return con;
    }

}

This class connects Java program to MySQL database.

8️⃣ Write INSERT Code (Create)

Add this inside Add Button Action.

try{

Connection con = DBConnection.getConnection();

String query = "INSERT INTO student(name,email,course) VALUES(?,?,?)";

PreparedStatement pst = con.prepareStatement(query);

pst.setString(1, txtName.getText());
pst.setString(2, txtEmail.getText());
pst.setString(3, txtCourse.getText());

pst.executeUpdate();

JOptionPane.showMessageDialog(this,"Record Inserted");

}catch(Exception e){
System.out.println(e);
}

This adds new student data.

🔟 Write UPDATE Code

Inside Update button:

try{

Connection con = DBConnection.getConnection();

String query = "UPDATE student SET name=?,email=?,course=? WHERE id=?";

PreparedStatement pst = con.prepareStatement(query);

pst.setString(1, txtName.getText());
pst.setString(2, txtEmail.getText());
pst.setString(3, txtCourse.getText());
pst.setInt(4, Integer.parseInt(txtID.getText()));
